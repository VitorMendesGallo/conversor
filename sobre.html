Sobre o Conversor de Moedas

O que é este projeto?

Este conversor de moedas foi desenvolvido para facilitar a conversão de diversas moedas estrangeiras para o Real brasileiro (BRL), utilizando cotações atualizadas em tempo real. O sistema é simples, moderno e responsivo, pensado para ser fácil de usar em qualquer dispositivo.

Funcionalidades

  • Conversão instantânea de várias moedas para Real
  • Cotações atualizadas automaticamente via API
  • Interface intuitiva e responsiva
  • Design limpo e agradável

Tecnologias utilizadas

  • HTML5, CSS3 (com Bootstrap) e JavaScript
  • Consumo de API pública para cotações
